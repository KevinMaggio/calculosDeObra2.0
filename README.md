# calculos De Obra
### Es una aplicacion creada para los obreros freelance, facilitandoles calculos matematicos largos. 
### La aplicacion cuenta con ventajas como el calculo de presupuestos , metros Cuadrados, cantidad de materiales etc... 

# Requisitos

### Podriamos decir q es apto para todo el mundo, aunque requerimos un android 5 o superior para correr la App.
